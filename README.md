# Core Payment Ledger & Wallet System

## Overview
A secure digital wallet system built using Spring Boot.

## Tech Stack
- Java
- Spring Boot
- PostgreSQL
- Redis
- React
- Maven

## Features (Planned)
- User Authentication (JWT)
- Wallet Management
- Deposit & Withdraw
- Peer-to-Peer Transfers
- Double-Entry Ledger
- Idempotency using Redis
- Transaction History

## Status
🚧 Project under development.
